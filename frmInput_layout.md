## バリデーション仕様

- **図番**（txtZuban）または**指示番号**（txtShiji）のどちらかは必須
- 両方とも空の場合はエラーメッセージを表示し、処理を中断する

---

## フォーム組み込み手順（VBAエディタ操作）

1. Excel VBAエディタを開く（Alt+F11）
2. プロジェクトエクスプローラで対象 `.xlsm` を右クリック
3. 「挿入」→「ユーザーフォーム」を選択
4. フォームの `Name` プロパティを `frmInput` に変更
5. 上記仕様に従いコントロールを配置（ツールボックスからドラッグ）
6. `frmInput.frm` のコードをフォームのコードウィンドウに貼り付け
7. `Module1.bas` の内容を Module1 のコードウィンドウに貼り付け（既存コードを置き換え）
8. ファイルを保存（`.xlsm` 形式）

---

## 処理フロー

```
① frmInput.Show
      ↓ OK押下
② 入力値 → 名前付き範囲へ書き込み
      ↓
③ CAM/WS/NC処理を実行
      ↓
④ スケジュールシートのデータ範囲テーブルに1行追記
```

## スケジュールテーブル転記列

| テーブル列名 | 転記元 |
|---|---|
| 登録日 | 実行日（Date関数） |
| コード番号 | Range("コード番") |
| 図番 | Range("図番") |
| 指示番号 | Range("指示番号") |
| 品名 | Range("製品名") |
| 部品番号 | Range("部品番号") |
